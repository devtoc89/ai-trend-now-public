# Storybook Application

UI 컴포넌트를 개발하고 문서화하기 위한 Storybook 기반의 개발 환경입니다.

## 주요 기능

- UI 컴포넌트 개발 및 테스트
- 컴포넌트 문서화
- 인터랙티브한 컴포넌트 미리보기
- 접근성 테스트
- 반응형 디자인 테스트

## 기술 스택

- Node.js (>=18)
- TypeScript
- Next.js
- Storybook 8
- Tailwind CSS
- Biome (코드 포맷팅)

## 시작하기

### 설치 및 실행
# 의존성 설치
pnpm install
# Storybook 개발 서버 실행 (기본 포트: 6006)
pnpm dev-story
# Storybook 빌드
pnpm build-story
# 빌드 결과물 제거
pnpm remove-build
